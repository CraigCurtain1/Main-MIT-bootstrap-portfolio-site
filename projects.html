Web Portfolio Examples

Here is a list of some examples websites I have built. Below are three projects presented as part of my portfolio for the Professional Certificate in Coding: Full Stack Development with MERN program by MIT XPro and Emeritus.


PacMen Factory

By clicking a button, users are able to generate a unlimited number of PacMen. From the classic iconic video game, and send them flying across the screen, chomping away just like in the video game. This project ahows use how to manipulate the DOM, use and work with timers, and implement edge detection.

Moving Eyes

On page load, a pair of eyes will greet the user, and the pupils will follow the mouse wherever it goes. This project shows the use of CSS grid, and the ability to programatically change styles based on user actions.
